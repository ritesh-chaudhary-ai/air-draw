📌 Overview

Air Draw is a browser-based computer vision application that allows users to draw in the air using real-time hand tracking through their webcam. The project uses MediaPipe Hands to detect finger movements and converts gestures into digital drawing.

---

✨ Features

- ✋ Real-time hand tracking
- 🎨 Air drawing using finger gestures
- 📷 Webcam support
- ⚡ Fast performance with Vite
- 🌐 Live deployment
- 💻 Responsive interface

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- MediaPipe Hands
- Vite
- Web Camera API
